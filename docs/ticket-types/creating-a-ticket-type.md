# Creating a Ticket Type

## Steps

1. Navigate to **Ticket Types** in the main navigation.
2. Click **New** (or the equivalent "New Ticket Type" button).
3. Fill in the form:

| Field | Notes |
|---|---|
| **Name** | Required. The display name for this type. |
| **Slug** | Auto-generated from the name. Can be edited manually. |
| **Description** | Optional. Describes when this type should be used. |
| **Default SLA Days** | Number of days before overdue. Leave blank for no SLA. |
| **Icon** | Identifier for a visual icon. |
| **Color** | Pick a color for the type badge. |
| **Sort Order** | Lower numbers appear first in the type selector. |
| **Active** | Toggle on to make this type available when creating tickets. |

4. Click **Save**.

The new type is immediately available in the **New Ticket** type selector (if set to Active).
