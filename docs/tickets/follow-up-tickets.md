# Follow-Up Tickets

A follow-up ticket is a child ticket linked to a parent. Use follow-ups to track related or dependent issues without losing the connection to the original complaint.

## Creating a Follow-Up

1. Open the parent ticket.
2. Click **Create Follow-Up** in the actions bar.
3. A new ticket form opens, pre-linked to the parent. Fill in the details as you would for any new ticket.
4. Submit the form.

## Viewing Linked Tickets

On any ticket's **Details** tab, the **Linked Tickets** section shows:

- **Parent**: if the current ticket is a follow-up, the parent ticket is shown with its reference number and title
- **Follow-Ups**: any child tickets that were created as follow-ups to the current ticket

Clicking any linked ticket reference navigates directly to that ticket.
