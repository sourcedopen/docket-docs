# Creating a Contact

## Steps

1. Navigate to **Contacts** in the main navigation.
2. Click **New Contact**.
3. Fill in the form:

| Field | Notes |
|---|---|
| **Name** | Required. The contact's full name or organisation name. |
| **Designation** | Optional. Job title or role. |
| **Organization** | Optional. The organisation this person belongs to. |
| **Email** | Optional. Contact email address. |
| **Phone** | Optional. Contact phone number. |
| **Address** | Optional. Physical address. |
| **Type** | Required. Authority, Company, Department, or Individual. |
| **Notes** | Optional. Any additional information about this contact. |
| **Tags** | Optional. Colour-coded labels for organisation. |
| **Documents** | Optional. File attachments (20 MB max per file). |

4. Click **Save**.

The contact is now available for selection when creating or editing tickets.
