# Attaching Documents

Documents can be attached to tickets, contacts, and comments. All file uploads have a maximum size of **20 MB per file**.

## Uploading During Ticket or Contact Create / Edit

When creating or editing a ticket or contact, a file input is available at the bottom of the form. You can:

- Select a single file or multiple files at once
- Upload different file types (PDFs, images, spreadsheets, etc.)

Click **Save** — the files are uploaded and associated with the record.

## Uploading via a Comment

Files can also be attached when [adding a comment](../comments-and-timeline/adding-comments.md) on a ticket. Use the file input in the Add Comment form.

## Viewing Attached Documents

- **Ticket** — documents are listed at the top of the **Timeline** tab
- **Contact** — documents are listed on the contact's detail page

Each document is shown as a downloadable link.

## Deleting a Document

Click the **Delete** button next to a document. A confirmation prompt is shown before the file is removed.
